Para ello, es necesario leer este documento previo antes de revisar el codigo desarrollado en Python.

    1. La estructura del codigo esta hecha en Python al 100% dentro de su desarrollo
    2. El codigo "main.py" contiene directamente un recopilatorio de todos los ejercicios, divididos en 2 ramas
        subidos en GitHub 
    3. Cada codigo de ejercicio es independiente pero dependiente a la vez del archivo "main.py" 
    4. El archivo "main.py" esta distribuido por funciones, las cuales en principio determinan el llamado a los otros    
        archivos .py relacionados, y en funcion al llado que haga el usuario para dicha ejecucion de acuerdo al ejercicio 
        de la maquina de Turin
    5. En principio, la estructura en main.py se desarrolla de forma sencilla para el entendimiento d elo que se quizo      realizar, puesto que al ejecutarse main.py, pide al usuario que digite el lenguaje y asi mismo, pedira que indique de 1 a 10 la solucion de acuerdo al metodo, siguiendo la estructura del documento
